# high-throughput
## DFT calculation
```
main_submissionfile and cp2k.inp are shell scripts
all .py files, main_submissionfile and cp2k.inp should be in the same location for DFT


DFT running process:
1. main.py: create input.xyz for different structures from the template
2. run_MXene.py: run the jobs
3. COOH_restart.py or restart.py (if there is MXene-RESTART.kp error )
4. get_results_dft.py
```

## delta_G calculation
```
main_submissionfile is shell scripts
cp2k.inp is generated by run.py
all .py files, main_submissionfile should be in the same location for DFT


delta_G running process:
1. cp_MXene-pos-1.py: get converged structure from DFT calculation
2. main.py: create input.xyz for different structures from MXene-pos-1
2. cp_run.py: copy run.py to sub-directories and run the jobs
3. del_empty.py: restart the job 
4. get_results_G.py: get delta_G results
